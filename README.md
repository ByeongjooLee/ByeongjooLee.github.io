## 구조

장식 없는 단일 페이지 정적 사이트입니다. 빌드 과정 없이 GitHub Pages가 파일을 그대로 서빙합니다.

- `index.html` — 페이지 전체 (내용 + 스타일)
- `photo.jpg` — 프로필 사진
- `cv.pdf` — 이력서 PDF

내용을 고치려면 `index.html`의 해당 부분 텍스트만 수정하면 됩니다.
